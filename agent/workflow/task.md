# Short Code: /task
**Trigger:** When user inputs `/task [description]`.

## Task Execution Protocol
1. **Rule Check 1:** Read `agent/rules/development.md`.
2. **Analysis:** Break down the task.
3. **Working Log:** Create `docs/working-logs/YYMMDD_TaskName.md`.
4. **Plan:** Propose implementation plan.
5. **Approval:** Wait for user approval before coding.
